## 📌 Project Overview
This repository contains a comprehensive statistical analysis workflow developed during a **Coursera Guided Project**. The project focuses on applying **Six Sigma methodologies** to healthcare data, specifically analyzing patient "Time to Admit" using the R programming language.

## 🛠️ Key Features & Tasks
The project is divided into specialized modules to ensure data quality and process capability:

* **Task 1: Data Wrangling** – Importing datasets, subsetting by category (e.g., Minor Injury), and inspecting data structures.
* **Task 2: Descriptive Statistics** – Calculating Centering (Mean, Median, Mode) and Spread (Standard Deviation, Variance, IQR, SPAN).
* **Task 3: Statistical Sampling** – Demonstrating the Central Limit Theorem by collecting 1,000 samples and calculating 95% Confidence Intervals.
* **Task 4: Data Visualization** – Creating professional Histograms with Normal Curve overlays, Boxplots by category, and Pareto Charts.
* **Task 5: Probability Distributions** – Generating and analyzing Uniform, Normal, Exponential, and Log-Normal data.
* **Task 6: Distribution Fitting** – Using Anderson-Darling and Kolmogorov-Smirnov tests to identify the best-fit model for the data.
